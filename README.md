## build (node version >= 12)

1. install deps
    ```
    npm install
    ```
1.  compile saito
    ```
    npm run nuke 
    ```
    Might take a while. Wait till `Bundle Success!`
1.  run local dev
    ```
    npm start
    ```
    http://localhost:12101/
