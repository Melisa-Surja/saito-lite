About Saito

Saito is a solution to the problem of how to build a blockchain that preserves trustlessness at scale.

We prioritize building a real community with organic transactions and applications users want to use. The useability and usefulness of the network always takes priority. Saito is a powerful tool to build peer to peer applications, share messages and data securely, even transact other crypto.

Saito is an open network and users always own their data. We encourage anyone interested to get involved.
